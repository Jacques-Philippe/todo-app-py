# get set up (dev)
1. Install python3
1. Clone the repo
1. python3 -m virtualenv .venv
1. Start the VM
    - Mac: `source .venv/bin/activate`
    - Windows: `.\.venv\bin\activate.bat`
1. pip install -r requirements.txt
